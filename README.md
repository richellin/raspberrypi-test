# raspberrypi-test
### Use
+ Raspbian GNU/Linux 8
+ Python 2.7.9
+ gcc version 4.9.2 (Raspbian 4.9.2-10)
+ wiringPi
+ gpio version: 2.32

### C
```sh
gcc -o gpio-test1 gpio-test1.c -lwiringPi
sudo ./gpio-test1
```

### Python
```sh
sudo python gpio-test1.py
```
